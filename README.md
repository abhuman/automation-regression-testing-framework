# Automation & Regression Testing Framework

## Role: Software Tester / QA Engineer

This repository demonstrates an automation and regression testing framework designed to support repetitive test execution, failure analysis, and test coverage improvement for enterprise web applications.

## Responsibilities Covered
- Supported automation test scripts for repetitive regression scenarios
- Executed automated test runs and analyzed failures
- Improved test coverage and reduced manual testing effort

## Framework Highlights
- Selenium with Python
- Pytest-based test execution
- Page Object Model (POM)
- Modular and scalable design
- Regression-ready structure

## Scope of Automation
- Login workflows
- Dashboard navigation
- Smoke and regression test suites

> Application name is intentionally omitted to maintain confidentiality.
